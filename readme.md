# Aradhana - Professional Portfolio Website

A sleek, ultra-minimalist, dark-themed personal portfolio blueprint designed for a **Computer Science Engineer & Agile Project Coordinator**. Built from scratch using modern web standards, this responsive webpage showcases the unique combination of AI-driven engineering workflows and scalable agile project delivery frameworks.

---

## 🚀 Live Preview & Aesthetics
* **Theme:** High-end Corporate Tech Dark Mode (`#060913`)
* **Typography:** Plus Jakarta Sans
* **Layout Systems:** CSS Grid (for modular alignment) & Flexbox (for responsive content control)
* **Design Paradigms:** Clean visual hierarchies, interactive button states, and separate competency pill tags.

---

## 🛠️ Tech Stack & Architecture

* **Frontend Structure:** Semantic HTML5 to ensure optimal SEO performance and screen-reader accessibility.
* **Layout Styling:** Modern vanilla CSS3 utilizing custom properties (CSS variables), CSS Grid (2x2 adaptive configuration), and global transitions. No bloated libraries or external frameworks required.
* **Logic Framework:** Light, zero-dependency Vanilla JavaScript (ES6+) managing system events and handling secure form operations.

---

## 🌟 Features Mapped to My Resume

* **Hero Header Section:** Highlights British Council Certified B2 CEFR English proficiency and velocity-driven engineering metrics.
* **Core Competencies Matrix:** Dedicated 3-column structural layout dividing skills into AI-Driven Engineering, Development & Design, and Agile Project Coordination.
* **Featured Projects Pipeline:** An adaptive, hover-reactive grid featuring deep-dives into data parsing pipelines, frontend tools, and cross-functional Agile operational templates.
* **Interactive Contact Form:** A fully operational form with custom validation styling, real-time element focus states, programmatic state resets, and structured data tables printed via console hooks.
